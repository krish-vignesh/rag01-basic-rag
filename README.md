# RAG01 - Basic RAG System

A Retrieval-Augmented Generation (RAG) chatbot built using:

- LangChain
- ChromaDB
- HuggingFace Embeddings
- Ollama
- Llama 3.2
- RAGAS

## Features

- PDF ingestion
- Text chunking
- Vector embeddings
- Semantic retrieval
- Question answering
- RAG evaluation
